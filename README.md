# NoSafeExit

Top-down extraction shooter prototype made with Unity.

## Latest Release
- v0.2.0 Gameplay Update
- https://github.com/Hanming1008/NoSafeExit/releases/tag/v0.2.0

## Current Features
- Player movement with sprint + stamina drain/regen
- Player and enemy health system with death handling
- Mouse-accurate player facing
- Crosshair cursor in gameplay
- Shooting, hit processing, projectile visuals, and shell ejection
- Extraction zone countdown and extraction success flow
- Player movement lock after successful extraction
- Ground snap workflow for gameplay objects (player/enemy/loot/extraction zone)

## Project Status
- Playable prototype under active development
- Current focus: combat feel, map iteration, and extraction gameplay loop polish
